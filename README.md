### WarBlog

A dead-simple plugin to edit and publish your Astro blog.

### Setup

1. **Configure Settings**
   - Open Settings â†’ WarBlog
   - Browse to select your Astro project root
   - Set a password for blog updates
   - Customize content folder path
   - (optional) Customize the name of the folder for the blog in your vault

2. **Edit Your Blog**
   - Your blog posts will appear in the configured folder (default: "Blog")
   - Edit `.mdx` files directly in Obsidian

3. **Publish Changes**
   - Run command: **"Update blog to most recent changes"** (Cmd/Ctrl + P)
   - Enter your password
   - Done! Changes are committed and pushed to git
